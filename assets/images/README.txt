Drop-in real photos
-------------------
If you prefer real photographic images instead of the built-in SVG illustrations, add JPG/PNG files here and update the <img> src attributes in the HTML files.

Suggested files:
- hero.jpg (dove or candle)
- gandhi.jpg (Salt March, public domain or Creative Commons)
- mlk.jpg, mandela.jpg, malala.jpg, dalai-lama.jpg (ensure license allows reuse)
